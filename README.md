## ✦ What It Does

CFO Copilot is a full AI-powered financial operating system for Indian startups. It replicates every core function of a CFO:

### 1. 📊 Real-Time Financial Dashboard
- Live runway calculation (Cash ÷ Monthly Burn)
- Burn rate tracking with month-over-month trends
- MRR, cash balance, NRR — all in one view
- 6-month revenue vs burn chart
- Instant alerts when runway drops below 12 months

### 2. ✦ AI CFO Chat (Powered by Claude)
Ask anything in plain English:
- *"Can I afford to hire 3 engineers this month?"*
- *"When should I raise my next round?"*
- *"Do I have any GST compliance issues?"*
- *"What's my burn rate trend?"*

The AI has full context of your company's financials and responds with real numbers, not generic advice.

### 3. 📋 Automated Financial Statements
Auto-generated from your data — no manual work:
- **P&L Statement** — Revenue, COGS, Gross Profit, EBITDA, Net Profit
- **Balance Sheet** — Assets, Liabilities, Equity
- **Cash Flow Statement** — Operating, Investing, Financing activities
All exportable as text files.

### 4. 🛡️ Compliance Calendar (India-Specific)
- Auto-generates GST and TDS filing deadlines based on your founding date
- Tracks GSTR-3B, TDS quarterly returns
- Shows days overdue + penalty amounts
- Compliance health scores per category
- Flags overdue filings on your fundraising score

### 5. 🎯 Fundraising Intelligence
- **VC Readiness Score** (/100) — know if you're Series A ready before walking into a meeting
- Top 3 things to fix before raising
- Auto-generated board deck (6 slides, real numbers)
- VC questions prep with model answers
- Term sheet generator with dilution calculator
- Investor update email auto-generated monthly

### 6. ⚡ Unit Economics Calculator
- CAC (Customer Acquisition Cost)
- LTV (Customer Lifetime Value)
- LTV:CAC ratio (benchmark: 3x+)
- CAC payback period
- Gross margin %
- Magic number
Color-coded against Indian SaaS benchmarks.

### 7. 🔭 Revenue Forecasting
- 3 models: Linear, Compound Growth, Conservative
- 6-month MRR projection with visual chart
- Growth rate analysis from historical data
- ARR projections with Series A milestone tracking

### 8. 🏦 Cap Table & Dilution Modeling
- Add founders, investors, ESOP pool
- Full ownership percentage table
- Dilution simulator — see exactly how a new round affects everyone
- Post-money valuation calculator

### 9. 👥 Real Payroll Engine (India Compliant)
- Add employees with salary
- Auto-calculates **PF** (12% of basic, up to ₹15K)
- Auto-calculates **ESI** (3.25% if salary ≤ ₹21K)
- Auto-calculates **TDS** (10% above ₹50K/month)
- Shows gross salary, net take-home, and cost-to-company per employee

### 10. 🔮 Scenario Modeling
- Hiring impact simulator — model runway before posting the job
- Revenue scenario builder (sliders for hires, marketing, growth rate, churn)
- Presets: Base Case, Aggressive, Conservative
- Instant recalculation of runway and cash flow positive date

### 11. 🧾 Invoice Tracking & Reconciliation
- Add and track invoices by client
- Mark paid / pending / overdue
- Receivables metrics: total invoiced, received, pending, overdue
- Bank statement vs invoice reconciliation
- Auto-generated double-entry journal entries

### 12. 🔌 Data Integrations
**Import from:**
- Bank statements (CSV/Excel) — HDFC, ICICI, Axis, Kotak, SBI, Yes Bank, IndusInd
- **Tally** — export CSV from Tally, import directly
- **Zoho Books** — export reports, import directly
- **HubSpot / Salesforce / Pipedrive** — CRM pipeline for revenue forecasting
- Any CSV/Excel spreadsheet — AI auto-maps columns

**Template downloads** for all integrations so founders know exactly what format to use.

### 13. 📈 Industry Benchmarks
Your metrics vs Indian SaaS benchmarks side by side:
- MoM Revenue Growth (Seed: 10–20%, Series A: 15–25%)
- Gross Margin (Seed: 60–70%, Series A: 70–80%)
- NRR (benchmark: 100–110% for Series A)
- Burn Multiple (< 1.5x for Series A)
- Runway (18–24 months for Series A)
- Headcount efficiency (₹/person MRR)

### 14. 🔍 Audit Trail & Fraud Detection
- Every data change logged with timestamp and user
- Auto-flags anomalies: burn doubling, revenue dropping 50%+, unexpected cash spikes
- Full audit log for due diligence

### 15. 👥 Team Access & Roles
- Invite team members by email
- 4 roles: **Admin** (full), **Finance** (view + edit), **CA/Auditor** (view + compliance), **Investor** (dashboard only)
- Permission matrix per role
- Full audit trail of who changed what

### 16. 🔔 Push Notifications
- Browser push notifications for filing deadlines
- Runway warnings (below 12 months)
- WhatsApp alerts (coming soon)

## 🏗️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript (ES6+) |
| **Fonts** | Syne (headings) + JetBrains Mono (numbers) |
| **Backend / Database** | Supabase (PostgreSQL + Auth + Row Level Security) |
| **AI** | Anthropic Claude (via Supabase Edge Functions) |
| **File Processing** | SheetJS (xlsx) for CSV/Excel import |
| **Payments** | Razorpay (India) |
| **Hosting** | GitHub Pages (current) → Vercel (planned) |
| **Auth** | Supabase Magic Link (passwordless email) |

### Why This Stack?

- **No framework** — pure HTML/JS loads in under 1 second, works on 2G connections common in Tier 2/3 India
- **Supabase** — open source Firebase alternative, Row Level Security ensures data isolation between companies
- **Claude API via Edge Functions** — API key never exposed to frontend, usage metered per plan
- **GitHub Pages** — zero infrastructure cost during early stage, can migrate to Vercel without code changes

## 🗄️ Database Schema

```sql
-- Companies table
CREATE TABLE companies (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id     UUID REFERENCES auth.users(id),
  name        TEXT NOT NULL,
  stage       TEXT, -- Bootstrapped, Pre-Seed, Seed, Series A, Series B+
  industry    TEXT,
  founded     TEXT, -- YYYY-MM format for compliance calculations
  gstin       TEXT,
  created_at  TIMESTAMPTZ DEFAULT now()
);

-- Financial snapshots (one row per company per month)
CREATE TABLE financial_snapshots (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id  UUID REFERENCES companies(id),
  month       TEXT NOT NULL, -- YYYY-MM format
  mrr         NUMERIC,       -- Monthly Recurring Revenue (₹ Lakhs)
  burn        NUMERIC,       -- Total monthly burn (₹ Lakhs)
  cash        NUMERIC,       -- Cash in bank (₹ Thousands = cash*100 = ₹ Lakhs)
  salaries    NUMERIC,       -- Salary expense (₹ Lakhs)
  cloud       NUMERIC,       -- Cloud/infra expense (₹ Lakhs)
  marketing   NUMERIC,       -- Marketing spend (₹ Lakhs)
  headcount   INTEGER,       -- Team size
  customers   INTEGER,       -- Paying customers
  nrr         NUMERIC,       -- Net Revenue Retention %
  created_at  TIMESTAMPTZ DEFAULT now()
);

-- Subscriptions / usage tracking
CREATE TABLE subscriptions (
  id                 UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id            UUID REFERENCES auth.users(id),
  plan               TEXT DEFAULT 'free', -- free, starter, pro, enterprise
  ai_messages_used   INTEGER DEFAULT 0,
  reset_date         TIMESTAMPTZ,
  created_at         TIMESTAMPTZ DEFAULT now()
);

-- Row Level Security (all tables)
ALTER TABLE companies ENABLE ROW LEVEL SECURITY;
ALTER TABLE financial_snapshots ENABLE ROW LEVEL SECURITY;
ALTER TABLE subscriptions ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users own their companies"
  ON companies FOR ALL USING (auth.uid() = user_id);

CREATE POLICY "Users own their snapshots via company"
  ON financial_snapshots FOR ALL
  USING (company_id IN (SELECT id FROM companies WHERE user_id = auth.uid()));

CREATE POLICY "Users own their subscription"
  ON subscriptions FOR ALL USING (auth.uid() = user_id);
```

## 🚀 Supabase Edge Function (AI Chat)

The AI is powered by a Supabase Edge Function that keeps the Anthropic API key server-side:

```typescript
// supabase/functions/ai-chat/index.ts
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"

const ANTHROPIC_KEY = Deno.env.get('ANTHROPIC_API_KEY')
const PLAN_LIMITS   = { free: 5, starter: 100, pro: 999999, enterprise: 999999 }

serve(async (req) => {
  const { messages, system, user_id } = await req.json()

  // Check usage limits from database
  // ... (see full implementation in /supabase/functions/ai-chat/index.ts)

  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'x-api-key': ANTHROPIC_KEY,
      'anthropic-version': '2023-06-01',
      'content-type': 'application/json',
    },
    body: JSON.stringify({
      model: 'claude-opus-4-5',
      max_tokens: 400,
      system,
      messages
    })
  })

  // Return reply + updated usage count
})
```

**Deploy:**
```bash
supabase functions deploy ai-chat
supabase secrets set ANTHROPIC_API_KEY=your_key_here
```

## 🔧 Local Development

Since this is a single-file HTML app, local development is simple:

### Option 1: VS Code Live Server
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Right-click `app.html` → **Open with Live Server**
3. Opens at `localhost:5500`

### Option 2: Python HTTP Server
```bash
python -m http.server 8000
# Open: http://localhost:8000/app.html
```

### Option 3: Node HTTP Server
```bash
npx serve .
# Open: http://localhost:3000/app.html
```

> ⚠️ **Important:** The app must be served from a server (not opened as a `file://` URL) because Supabase JS requires HTTP/HTTPS to initialise.

Click **"Try Demo — No Signup"** on the login screen to explore all features with pre-loaded data for "ZeroPeak Labs" (a fictional Seed-stage SaaS startup). No account required.

## ⚙️ Configuration

### Supabase Setup
1. Create a project at [supabase.com](https://supabase.com)
2. Run the SQL schema above in the SQL editor
3. Enable Email Auth with Magic Links
4. Set the redirect URL to your app URL
5. Replace in `app.html`:
```javascript
const SB_URL = 'https://YOUR_PROJECT.supabase.co';
const SB_KEY = 'your_publishable_key_here';
```

### Razorpay Setup
1. Create account at [razorpay.com](https://razorpay.com)
2. Get your API key from Dashboard → Settings → API Keys
3. Replace in `app.html`:
```javascript
key: 'rzp_live_YOUR_KEY_HERE'
```

### Anthropic API (Claude)
1. Get API key from [console.anthropic.com](https://console.anthropic.com)
2. Add to Supabase Edge Function secrets:
```bash
supabase secrets set ANTHROPIC_API_KEY=sk-ant-...
```
